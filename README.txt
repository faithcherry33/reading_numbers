한글 수 읽기 대결 - 최종 기능 통합본

교체/추가할 파일
1. app.js: 기존 app.js 전체 교체
2. index.html: 기존 index.html 전체 교체
3. battle-final.css: 저장소 루트에 새 파일로 추가

- firebase-config.js: 기존 Firebase 설정을 그대로 사용합니다.

포함 기능
- 대결 제안, 제안자 자동 참여
- 참여/참여 취소, 오른쪽 목록 참여 상태 표시
- 현재 접속 중인 수락자 4명 이상일 때만 시작
- 혼자하기 중 방해하지 않는 초대 안내
- 시작 순간 참여자 명단 고정, 뒤늦은 합류 차단
- 방 나가기/대결 퇴장/브라우저 뒤로가기 확인 및 즉시 퇴장 처리
- 새로고침·탭 종료·연결 끊김 시 즉시 퇴장하지 않고 동일 UID 재접속 허용
- 같은 UID의 이전 대결 자동 복귀
- 닉네임이 달라졌을 때 이전 닉네임으로 이어하기 또는 이전 대결 포기 선택
- 같은 UID가 여러 대결에 동시에 참여하는 것 차단
- 제안자가 모집 중 의도적으로 나가면 제안 즉시 취소
- 제안자가 갑자기 끊기면 3분 유지 후 제안 자동 삭제
- 마지막 답 제출 후 3분 무입력 자동 퇴장
- 대결 종료까지 한 번도 제출하지 않은 참가자 미참여 표시
- 전원 무입력/탈락/중도 퇴장 시 자동 종료
- 전날 남은 오래된 대결 및 참가자 0명 비정상 대결 정리
